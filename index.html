what is a manifesto?

hosting your website

1. Sign up for GitHub (if you haven't already)

2. Create a new repository on GitHub

3. Copy the HTML content Claude generated for you

4. Paste the HTML into a new file named 'index.html' in your repository

5. Go to your repository's "Settings" tab

6. Scroll down to "GitHub Pages" section

7. Under "Source", select "main" branch

8. Click "Save"

your website is now live at username.github.io/repository-name!
